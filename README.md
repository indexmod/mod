# mod — базовый концепт живой вики на Cloudflare Workers

## 📌 Описание проекта

**mod** — это минималистичная серверная вики-система, построенная на Cloudflare Workers + KV Storage.

Проект реализует концепцию «живого документа», где:

* страница редактируется прямо в браузере
* изменения сохраняются автоматически
* нет классической CMS или панели администратора
* данные хранятся в KV (Cloudflare)

---

## 🧠 Архитектура

Проект состоит из трёх основных частей:

### 1. Worker (worker.js)

Cloudflare Worker выполняет:

* отдачу HTML интерфейса
* отдачу CSS
* API для чтения/записи документов

### 2. KV Storage (DOC)

Используется как база данных:

* ключ: `home`
* значение: текст страницы

### 3. Frontend (встроенный в Worker)

* contenteditable редактор
* autosave через debounce
* загрузка и сохранение через fetch API

plain text

```
ok
```

---

## 🖥 Интерфейс

При открытии `/`:

* загружается редактор
* автоматически подтягивается содержимое `/doc/home`
* изменения сохраняются через 200ms debounce

---

## 💡 Концепция проекта

Проект реализует идею:

> «интерфейс = документ = база данных»

Это означает:

* нет разделения между UI и storage
* страница является источником истины
* редактирование происходит напрямую в живой среде

## 🚀 Возможные расширения (форк)

Проект задуман как базовый слой для дальнейшего развития:

### 1. Мультистраничность

* `/doc/:slug`
* граф страниц

### 2. Graph-структура

* связи между страницами
* wiki-граф как база данных

### 3. Real-time sync

* WebSocket / Durable Objects
* мульти-пользовательское редактирование

### 4. Версионность

* история изменений
* откат состояний

### 5. Rich editor

* markdown парсер
* блоки (notion-like)

---

## 🧩 Текущие ограничения

* нет авторизации
* нет версионности
* одна фиксированная страница (`home`)
* нет конфликт-резолвинга
